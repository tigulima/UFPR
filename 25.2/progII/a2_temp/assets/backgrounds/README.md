# Diretório de Backgrounds

Coloque aqui os backgrounds das fases.

## Backgrounds Necessários (Requisitos Mínimos)

- `level1.png` - Background da fase 1

## Funcionalidades Extras

- `level2.png` - Background da fase 2 (funcionalidade extra [5])

## Formato Recomendado
- PNG ou JPG
- Resolução: Maior que 800x600 para suportar rolling background
- Recomendado: 3000x600 ou maior para fases longas
