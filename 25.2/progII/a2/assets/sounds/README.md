# Diretório de Sons

Coloque aqui os arquivos de áudio do jogo.

## Sons para Funcionalidade Extra [10]

- `menu_music.ogg` - Trilha sonora do menu
- `level_music.ogg` - Trilha sonora da fase
- `jump.wav` - Efeito sonoro de pulo
- `damage.wav` - Efeito sonoro de dano
- `collect.wav` - Efeito sonoro de coleta de item

## Formato Recomendado
- OGG para músicas (menor tamanho, boa qualidade)
- WAV para efeitos sonoros curtos
