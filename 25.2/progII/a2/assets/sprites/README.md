# Diretório de Sprites

Coloque aqui os sprites do jogo.

## Sprites Necessários (Requisitos Mínimos)

### Personagem
- `player_idle.png` - Personagem em pé
- `player_walk_*.png` - Frames de animação andando
- `player_jump.png` - Personagem pulando
- `player_crouch.png` - Personagem abaixado
- `player_swing.png` - Personagem em ação de interação/desvio

### Obstáculos (mínimo 6 tipos)
- `obstacle_static_*.png` - Obstáculos estáticos (espinhos, buracos, etc)
- `obstacle_moving_*.png` - Obstáculos em movimento (troncos, animais, etc)
- `platform_*.png` - Plataformas móveis

## Formato Recomendado
- PNG com transparência
- Resolução adequada (exemplo: 64x64 para personagem, variado para obstáculos)
